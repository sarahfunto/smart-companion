# 🎙️ Smart Companion

**Smart Companion** is an MVP (Minimum Viable Product) built with Streamlit. It materializes Limor's conceptual architecture for semantic and psychological client analysis.

## 🚀 Key Features
* **Real-time Slot Capture:** Automatic identification of client roles, constraints, technology stacks, and growth objectives.
* **Psychological Profiling (Backend Insights):** Real-time tracking of underlying client anxieties (*Fears*) and cognitive preferences (*Lens*).
* **Strategic Diagnostics:** Automated generation of customized recommendations and tailored sales approach tips for the POC.

## 🛠️ How to Use
The application includes a live voice recognition feature (Microphone button) as well as a text-input backup field (*Backup*) to ensure a smooth and seamless product demonstration.
